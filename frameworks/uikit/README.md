# UIkit (local static bundle)

## Structure

- `dist/uikit.min.css`
- `dist/uikit.min.js`
- `dist/uikit-icons.min.js`
- `examples/index.example.html`

## How to use

```html
<link rel="stylesheet" href="./uikit.min.css" />
<script src="./uikit.min.js"></script>
<script src="./uikit-icons.min.js"></script>
```
