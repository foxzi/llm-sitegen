# Water.css (local static bundle)

## Structure

- `dist/water.min.css`
- `dist/water-dark.min.css`
- `examples/index.example.html`

## How to use

Light theme:

```html
<link rel="stylesheet" href="./water.min.css" />
```

Dark theme:

```html
<link rel="stylesheet" href="./water-dark.min.css" />
```
